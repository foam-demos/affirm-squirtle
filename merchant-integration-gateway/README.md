# Merchant Integration Gateway

Partner API gateway powering integrations with Amazon, Shopify, Stripe, and 290K+ enterprise merchants. Handles webhook processing, OAuth flows, merchant onboarding, and API versioning. Built with Python/Flask and deployed across multiple AWS regions for low-latency partner callbacks.

**Stack:** Python 3.11, Flask, Redis, MySQL, AWS Lambda

**Run locally:** `docker-compose up -d && flask run --port 8002`
